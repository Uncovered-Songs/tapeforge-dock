# TapeForge Dock

The online operating platform for **TapeForge**.

Dock is where you run your TapeForge environment from the browser:

- **Sessions** — see all your sessions in one place.
- **Configurations & data** — review and manage your setup and stored data.
- **Documentation & FAQs** — full reference and answers to common questions.
- **CREW** — the built-in assistant that helps you work the system.

> Status: early scaffolding. More to come.
